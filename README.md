# 🚛 Платформа для отслеживания грузоперевозок

## 🎯 Цель системы

Созить простую и прозрачную систему взаимодействия между:

* **Грузовладельцем (B2B, веб-платформа)**
* **Грузоперевозчиком / Водителем (мобильное приложение)**

С фокусом на:

* реальное GPS-отслеживание
* простой workflow
* прозрачные статусы
* минимальный friction

---

# 🖥 1️⃣ Веб-платформа (для грузовладельца)

## 🔐 Регистрация и вход

Грузовладелец регистрируется через:

* Email
* Телефон
* Google (опционально)

---

## 📦 Управление грузами

### 1. Главная панель (Dashboard)

Разделы:

* 🔵 Активные грузы
* 🟢 Завершённые грузы
* 🟡 Ожидающие принятия
* 🔴 Отменённые

---

## 📍 Детали активного груза

При нажатии на груз отображается:

* 📍 Точка отправления
* 🚛 Текущее местоположение водителя (в реальном времени)
* 🏁 Точка назначения
* 🗺 Карта с маршрутом
* ⏱ Статус (Назначен / Принят / В пути / Завершён / Подтверждён)
* 🕒 История движения (опционально, premium-фича)

---

## ➕ Создание нового груза

При создании указываются:

* Точка отправки
* Точка назначения
* Описание груза
* Дата и время
* Дополнительные комментарии

---

## 👤 Назначение водителя

Варианты назначения:

1. По ID
2. По телефону
3. По email
4. Через специальную ссылку-приглашение

---

## 👥 Раздел "Мои водители"

Грузовладелец может:

* Добавлять водителей в список
* Видеть их статус (онлайн / офлайн)
* Назначать груз в 1 клик
* Видеть историю работы по каждому водителю

---

# 📱 2️⃣ Мобильное приложение (для водителя)

## 🔐 Регистрация

Водитель регистрируется через:

* Телефон (OTP)
* Email

---

## 🏠 Главный экран

### 🔥 Hero Block (если есть активный груз)

Появляется автоматически сверху.

Содержит:

* Информацию о грузе
* Статус
* Кнопки действий

---

## 📦 Workflow водителя

### 1️⃣ Назначен груз

Водитель получает:

* Push уведомление
* Груз появляется в Hero Block
* Кнопка: **Принять**

---

### 2️⃣ После принятия

Появляется кнопка:
➡️ **Начать поездку**

---

### 3️⃣ После нажатия "Начать"

* Статус: В пути
* Активируется GPS tracking
* Начинается отправка координат
* Появляется кнопка: **Завершить**

---

### 4️⃣ Завершение поездки

После нажатия:

* Статус: Завершено водителем
* GPS tracking останавливается
* Отправляется:

  * SMS / Email уведомление грузовладельцу
  * Push на веб

---

# 🖥 Подтверждение грузовладельцем

На вебе появляется кнопка:

✔ **Подтвердить выполнение**

После подтверждения:

* Статус: Выполнено
* Груз переносится в архив

---

# 🔄 Полный жизненный цикл груза

```
Создан →
Назначен →
Принят →
В пути →
Завершён водителем →
Подтверждён грузовладельцем →
Архив
```
